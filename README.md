# Starter Project Nexys

## How to run Development

Install it and run:

```bash
yarn
yarn dev
```

## How to run Production

Install it and run:

```bash
yarn
yarn build
yarn start
```

## Storybook

```bash
yarn storybook
```

## Features

- Next.js
- Typescript
- AntDesign
- Formik
- React Query
- Yup
- Storybook
- Jest
- Prettier
- Eslint
